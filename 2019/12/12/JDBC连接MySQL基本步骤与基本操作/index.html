JDBC连接MySQL基本步骤与基本操作

1.加载数据库驱动程序

Class.forName(“com.mysql.jdbc.Driver”);

2.同数据库建立连接

Connection con=DriverManager.getConnection(“jdbc:mysql://localhost:3306/数据库名”,”用户名”,”密码”);

3.创建Statement对象,执行SQL语句

Statement sta=con.createStatement();

增

int i=sta.executeUpdate(“insert into 表名(name,age)values(‘？’,？)”);

改

int i=sta.executeUpdate(“update 表名 set name=’？’ where name=’？’”);

删

int i=sta.executeUpdate(“delete from 表名 where age=？”);

4.处理从数据库返回的结果

out.print(i);

5.关闭

sta.close();

con.close(); 依次关闭
